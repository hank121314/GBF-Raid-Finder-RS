Granblue Fantasy Raid Finder Helm repository

Installation

helm repo add gbf-raid-finder-rs https://hank121314.github.io/GBF-Raid-Finder-RS

helm repo update

helm install gbf-raid-finder-rs

Note

  • You have to specify the host config and environments variable with --values

Environments variables

  • GBF_RAID_FINDER_LOG_PATH: log path for server log file, default is stdout
  • REDIS_URL: redis connection url
  • TWITTER_API_KEY
  • TWITTER_API_SECRET_KEY
  • TWITTER_ACCESS_TOKEN
  • TWITTER_ACCESS_TOKEN_SECRET
